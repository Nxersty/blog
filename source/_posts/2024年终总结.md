# 2024年终总结

我一直觉得年终总结怎么写都难免有些“流水账”的感觉，但是在写这一篇之前，我又查了年终总结的定义：**人们对一年来的工作学习进行回顾和分析，从中找出经验和教训，引出规律性认识，以指导今后工作和实践活动的一种应用文体。年终总结的内容包括一年来的情况概述、成绩和经验教训及今后努力的方向。**我想，既然是回顾和分析，那我或许可以放开一点，不必拘泥于框架文采，稍微啰嗦一点好像也没什么大碍了。

**毕竟无论怎样，我总希望把这一年的故事细细回顾，讲给自己，也讲给你们听。**


## 序章

2024年1月份，我回到了高中，探访那些曾经教导过我的老师，顺便参加北邮的宣讲会。和我的物理老师聊天时，说起近况，她的一句评价令我记忆犹新。

```
她说：“感觉你没有以前那么阳光了。”
```

我有些恍然。她提到的“以前”，是指高中时代的我吗？变成大学生的我，怎么会不如高中时阳光呢？

于是，我开始回望这一年，2024年，也是我真正意义上度过的大学生活的第一年，回想起来，看看我这一年到底过得如何。

## 第一篇章：旅途中的诗篇

<iframe frameborder="no" border="0" marginwidth="0" marginheight="0" width=300 height=100 src="https://www.joy127.com/url/118775.mp3">
</iframe>


```
“旅行最大的好处，不是能见到多少人，见过多美的风景，而是走着走着，在一个际遇下，突然重新认识了自己。”
```

前几天看了美团发布的年度报告，我才发现今年居然去了不少城市旅行（我一直以为自己是个不爱出门的人🌚），感觉我也太会哄自己开心了，总能带着一颗放松的心去探索世界😋

**今年第一站是1月份，刚考完试和同学一起去天津。**这次说起来还挺坎坷的，中途突然接到消息，要赶回北京打九价，再加上朋友们纷纷发烧，虽然只有3天，但是回来以后还是累得半死😰😰😰我不是第一次去天津了，算是旧地重游，但这次，我终于去了那些曾未涉足的著名景点，或许以后还会再来，我不会把天津当做旅游目的地。

**今年第二站是2月份，过年的时候去三亚家庭旅行。**对于这次旅行我甚至在没放寒假前就开始期待了，不仅是因为那是一个可以在冬天穿短袖的温暖天堂，更因为三亚在我看来有着独特的地位。2012年，我曾在三亚度过一个假期，由于当时只有7岁，许多细节早已印象模糊，但是那份快乐我却始终铭记，所以三亚在我心中，一直是“白月光”级别的旅游城市。

这次去三亚，我发现自己已不再像7岁时那般沉浸在旅行的纯粹喜悦中。或许是因为我已经长大，对很多事物的兴趣与看法有所不同，又或许是回忆给那一次旅行加了分。不过虽然如此，这次还是很开心，给我的甲辰年开了个好头。

![三亚](https://s2.loli.net/2024/12/21/mJMoIyv6xZiHKpR.jpg)

**今年第三站是6月份，为了看蔡依林演唱会去济南。**本来我没打算游览济南（时常因为自己太没力气而放弃旅行），奈何同伴是行动派，我就在他的带领下，把大明湖、趵突泉等有名的景点一天速通完毕，晚上则沉浸于演唱会的热烈氛围，这次更是累得够呛。好在同伴是优秀的摄影师，拍出了不少我满意的照片（真的超级喜欢下面那一张啊），感谢感谢🤩

![最喜欢的一张](https://s2.loli.net/2024/12/21/nq1hlJEQdCeswbo.jpg)

Jolin饱饱更是美神降临🥰🥰🥰🥰🥰🥰

![Jolin](https://s2.loli.net/2024/12/21/lgiqAv48IswczEd.jpg)

**今年第四站是6月底，带着社会实践团队前往云南的西双版纳和昆明。**曾经我总能在沈石溪的小说里面读到他在西双版纳研究野生动物，小时候我就一直想亲眼去那里看看，所以这次去西双版纳也算是圆梦了吧。虽然没有看到野生动物，但是玩了超刺激的丛林飞跃，吃了著名的见手青，尝了正宗的云南过桥米线~~（不好吃，感觉不如河北版云南过桥米线）~~，也把西双版纳逛得差不多了。

**今年第五站是8月份，和家人一起去大连。**大连给我印象最深的是，我花了900r染了发，现在后悔极了（为什么要漂头发呢？我以后再也不漂了呜呜呜，我要我的大波浪😭😭😭）至于看海玩水什么的，我确实不像小时候那么喜欢了。

**今年最后一站是国庆，和朋友去南京。**南京很好，下次不来了，人多的挤死我了。。。这次旅行真是纯欢乐啊，两个抽象女大就有讲不完的乐子和接不完的梗，还有发不完的漂亮照片🤩顺带一提，鸭血粉丝汤真是太好喝了。

放张合照吧，虽然没有经过朋友的同意（**饱饱你不许不同意**😘）

![南京](https://s2.loli.net/2024/12/22/GmiyPwXdHB8Kh2T.jpg)

每个目的地都是一段故事，每一段旅程都是心灵的一次洗礼。回望这一年，旅行成了我生活中的一部分，尽管有时候只是短暂的逃离，但每一次出行都让我收获了成长和快乐。这想必是出游的意义吧，我在谱写这一年最精彩快乐的部分——旅行中的诗篇。

2024年，我有太多“第一次”做的事情了，回想起来能真切地感觉到，我确实是在这些“第一次”中成长了不少，还是挑比较重要的回顾一下吧。

**3月份的时候，第一次独自组织班级活动，带着同学们一起参观国家博物馆。**这次确实有很多青涩的地方吧，尤其是协调各方事宜时，确实有一些挑战。这些是以前没有经历过的，我在这次得到了锻炼，积累了经验，让我组织能力++。

有了这些经验，后面组织第二次团日活动时就顺利了很多，组织社会实践去云南的行程我也可以一手包办了。

**3月底，第一次真正和外国人面对面口语交流。**为了志愿时长，我去参加了全球研究生院巡展的北京站，没想到还获得了和那些招生办交流的机会。不试不知道，一试吓一跳，原来我的口语能力这么差啊，每说出来一句话之前要先在脑子里过一遍，导致思考时间太久，根本接不上话🥹当时我就决定以后苦练口语和听力，到现在已经做出了0个行动……

明年一定。

**5月份，第一次参加正式合唱比赛。**“五花”是我第一个合唱比赛，数来宝联队就是最牛的啊🫡为了准备这次合唱，我们排练时间着实不算短了，本来我就是想去混混分，结果发现这种正式合唱真好玩，比小时候的合唱好玩多了，明年还要来参加。五花的最后合影感觉拍的也不错，那时候还是卷发（呜呜呜呜呜呜大连你还我大波浪）

![五花合影](https://s2.loli.net/2024/12/21/qgxPRQDdHwhYZja.jpg)

**6月底社会实践，第一次给别人开讲座。**我们去云南也不是纯玩，还要在昆明实践，确实也做正经事了啊。我联系了昆明的一所小学，带着团队去给那里的学生进行了一次AI科普讲座。小朋友们太可爱了，讲得时候真的感觉不到一点紧张。见惯了开会时下面听众鸦雀无声，这里每个小孩都在认真听，活泼地给回应，实在是令我感动。本来一边社会实践一边出成绩很烦，这些小孩真是让我似人微活了🥹🥹🥹

![讲座照片](https://s2.loli.net/2024/12/21/YHn58IRViXbyZvA.jpg)

有了这次讲座的经验，后来优秀班集体答辩就更从容了。

## 终章

```
“当旧日的太阳落下，新的一轮正悄然升起。”
```

前面讲了很多，大部分是在讲故事，那现在到终章了，我总该讲点有意义的总结了。

让我们请出《光·白》好吗👏👏👏

回到开头那个问题，为什么作为大学生，反而没有高中时候的我“阳光”了呢？现在我有了答案。

我想应该有两个原因吧。

往好了想呢，就是我变成熟了，这是成长的标志，代表我开始脱离父母的照顾，尝试独立生活。

2024年认识了很多人，学会了好多事，积累了不少经验，我觉得自己就是比去年思维成熟，做事也变得稳重些。这样对比之下，大学生活就让我失去了以前的活力，比高中时候要考虑更多事情，不再像高中时那样充满冲劲。

往坏了想呢，就是没目标了。高中的时候目标多明确啊，我就要考xx分，就要考xx大学，我所做的一切都是为了高考，为了上更好的大学，有更好的未来。但是现在，我对于自己未来要何去何从，现在做的事情是为了什么，还处于迷茫阶段。

这个可能还要持续更久一点，要等到明年、后年，我才有可能明确自己的方向。有方向才有动力，有动力才有活力，有活力才会“阳光”起来啊。

对于目前来说，我的梦想生活可能就是每天躺在沙发上吃零食看电影打游戏（这个等到寒假就能实现），但这并不是一个永恒不变的愿望。至少回顾今年，我能告诉自己，我这一年没白活，还是有不少收获的。

 **”别想那么多，向前走就是了。“**

总结讲完了，讲讲2025的目标吧。

其实很简单：**那些想学但是被我拖延了很久的东西，慢慢地腾出时间来看一看吧；那些需要达成的标准，想办法搞定；明年写年终总结的时候，能像今年一样确信，这一年没白过。**

说起来容易，做起来难。我不奢望全部完成，我只希望明年能比今年更好。

最后呢，希望2025年对我好一点。
